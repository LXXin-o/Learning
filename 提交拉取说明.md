方式一：空白文件夹一键克隆
# 直接拉取完整仓库，自动生成项目文件夹
git clone git@github.com:LXXin-o/Learning.git

# 进入项目目录
cd Learning

方式 2：已有本地文件夹手动绑定
# 1.进入存放目录
cd D:\Learning
# 2.初始化本地Git仓库
git init
# 3.绑定SSH远程仓库
git remote add origin git@github.com:LXXin-o/Learning.git
# 4.拉取远程分支信息
git fetch origin
# 5.切换到主分支，下载全部代码文件
git checkout main

二、完整提交推送流程（本地改代码 → 上传 GitHub）
# 1.把所有修改文件标记待提交
git add .
# 2.本地保存改动，填写修改说明
git commit -m "这里写改动内容，例：新增测试代码"
# 3.上传到GitHub远程main分支
git push origin main

三、完整拉取更新流程（GitHub 有新代码 → 更新本地）
# 同步远程最新代码到本地main分支
git pull origin main
